# Homebrew Tap for bop

AI-powered code review tool with multi-provider LLM support.

## Installation

```bash
brew install delightfulhammers/tap/bop
```

## What's Included

- `bop` - Main CLI for running code reviews
- `bop-mcp` - MCP server for Claude Code integration
