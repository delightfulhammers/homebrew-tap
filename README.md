# Homebrew Tap for code-reviewer

AI-powered code review tool with multi-provider LLM support.

## Installation

```bash
brew install bkyoung/code-reviewer/code-reviewer
```

## What's Included

- `cr` - Main CLI for running code reviews
- `code-reviewer-mcp` - MCP server for Claude Code integration
